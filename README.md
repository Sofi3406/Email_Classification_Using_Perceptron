# 📧 Email Classification Using Perceptron

## Overview

This project classifies emails as **Spam** or **Not Spam (Ham)** using the Perceptron machine learning algorithm. It includes data preprocessing, model training, and model evaluation.

The project was developed using **Python** in **Jupyter Notebook (Anaconda)**.

## Technologies Used

* Python
* Jupyter Notebook (Anaconda)
* Pandas
* NumPy
* Scikit-learn
* Matplotlib

## Dataset

The project uses the **spam_Emails_data.csv** dataset.

## Project Structure

```text
Email-Classification-Using-Perceptron/
│

├── spam_Emails_data.csv

```

## Workflow

* Load the dataset
* Preprocess the email data
* Train the Perceptron model
* Evaluate the model
* Classify emails as Spam or Ham

## How to Run

1. Clone the repository.
2. Open the project in Jupyter Notebook or Anaconda.
3. Open **Email Classification Using Perceptron.ipynb**.
4. Run the notebook cells from top to bottom.
